# 雾霾森林 (Haze Forest)

一款抖音小游戏，玩家控制角色躲避和击杀怪物，收集经验和金币升级角色。

## 游戏特性

- **10个关卡** - 每关难度递增，Boss战
- **技能系统** - 20+种升级选择
- **商店系统** - 6款角色皮肤
- **无尽模式** - 通关后挑战极限

## 操作说明

- 滑动屏幕移动角色
- 角色自动攻击最近敌人
- 击杀敌人获得经验和金币
- 升级时选择强化技能

## 技术栈

- JavaScript
- Canvas 2D
- 抖音小游戏 API
