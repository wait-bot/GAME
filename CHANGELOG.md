### 目录结构

```
Haze forest/
├── game.js          # 游戏入口 (兼容模式)
├── game_full.js     # 完整游戏代码
├── src/
│   ├── config.js    # 配置文件
│   └── game.js      # 游戏主逻辑
├── icons/           # SVG图标 (20个)
│   ├── icon-atk.svg
│   ├── icon-hp.svg
│   └── ...
├── images/          # 游戏图片资源
├── backup/          # 原代码备份
└── README.md        # 游戏说明
```

### 使用方法

1. **使用完整版本** (推荐)
   - 将 `game_full.js` 重命名为 `game.js` 使用

2. **使用模块化版本**
   - 使用抖音小游戏原生模块加载方式
